Chrome 如何安装 BILIBILI-EX 插件 ​

本教程将指导您如何通过官方的 Chrome 应用商店 (Chrome Web Store) 安全、便捷地为您的 Google Chrome 浏览器安装 BILIBILI-EX 插件，以增强您的 Bilibili (B站) 使用体验。

先决条件 ​

  • 您需要安装 Google Chrome 浏览器。如果您尚未安装，请前往 Google Chrome 官方网站 下载并安装。

安装步骤 ​

  1. 访问插件页面: 点击下方的链接，或我们提供的其他官方链接，打开 BILIBILI-EX 在 Chrome 应用商店中的页面。

    点击这里前往 Chrome 应用商店安装 BILIBILI-EX

  2. 添加到 Chrome: 在插件页面中，点击右上角的 “添加到 Chrome” (Add to Chrome) 按钮。

  3. 确认安装: 浏览器会弹出一个确认窗口，请求插件所需的权限。请阅读权限说明，然后点击 “添加扩展程序” (Add extension) 按钮。

  4. 安装完成: 安装成功后，Chrome 浏览器的工具栏通常会显示 BILIBILI-EX 的图标。您可能需要点击浏览器右上角的拼图图标 (扩展程序) 来找到并固定 BILIBILI-EX 图标到工具栏上，方便使用。

  5. 开始使用: 现在您可以访问 Bilibili (bilibili.com) 网站，BILIBILI-EX 插件将自动生效，为您带来增强的浏览体验！

如果您在安装过程中遇到任何问题，请查阅 常见问题解答 或通过 联系我们 页面寻求帮助。
