常见问题 ​

BILIBILI-EX 是什么？ ​

BILIBILI-EX 是一款专为哔哩哔哩（Bilibili）网站设计的浏览器扩展插件，旨在提供更加流畅、舒适和个性化的B站使用体验。它集成了多种功能，包括封面提取、视频播放加速、首页视频刷新历史记录、功能增强等，让您在使用B站时更加便捷和高效。

BILIBILI-EX 支持哪些浏览器？ ​

我们支持所有主流的浏览器，包括：

  • Google Chrome
  • Microsoft Edge

我们建议您使用最新版本的浏览器，以确保所有功能正常运行。

BILIBILI-EX 是免费的吗？ ​

是的，BILIBILI-EX 是完全免费的。我们相信好的工具应该让更多人享受到，所以我们将一直保持核心功能的免费使用。

如果您喜欢我们的插件，可以通过赞助支持我们的开发工作，让 BILIBILI-EX 变得更好。

如何更新到最新版本？ ​

大多数浏览器会自动更新扩展插件，您无需手动操作。但如果您想确保使用的是最新版本，可以：

  • Chrome：访问 chrome://extensions/，开启 开发者模式，点击 更新 按钮
  • Edge：访问 edge://extensions/，检查是否有更新按钮
  • 离线版: 重新 下载最新版 直接安装替换即可.

我们会定期发布新功能和bug修复，建议保持更新以获得最佳体验。

为什么有些功能无法使用？ ​

如果您发现某些功能无法正常使用，可能有以下原因：

  • 浏览器版本较旧，不支持某些新特性
  • 与其他扩展程序冲突
  • B站网站结构变化，插件需要更新适配
  • 您的网络连接不稳定

请尝试更新浏览器和插件到最新版本，如果问题仍然存在，欢迎通过关于我们页面联系我们反馈问题。

BILIBILI-EX 会收集我的个人数据吗？ ​

我们高度重视用户隐私和数据安全。BILIBILI-EX 不会收集您的个人身份信息，所有功能都在本地运行，不会将您的浏览数据上传至任何服务器。

详细的 隐私策略 可以在我们的 隐私政策 页面查看。
