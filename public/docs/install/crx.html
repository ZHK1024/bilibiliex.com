如何离线安装 BILIBILI-EX 插件 (.crx 文件) ​

本教程适用于无法访问官方应用商店 (如 Chrome Web Store 或 Edge Add-ons) 或需要安装特定版本 BILIBILI-EX 插件的用户。通过下载 .crx 格式的插件文件，您可以手动将其安装到兼容的 Chromium 内核浏览器 (如 Google Chrome, Microsoft Edge 等) 中。

警告:

  • 安全风险: 请务必从官方来获取 .crx 文件。安装来源不明的插件可能存在安全风险。
  • 浏览器限制: 较新版本的 Chrome/Edge 可能会限制或阻止通过拖放 .crx 文件的方式安装非商店来源的扩展。如果遇到问题，您可能需要搜索特定浏览器版本的离线安装方法或考虑使用开发者模式加载解压后的插件。

先决条件 ​

  • 您需要安装兼容的 Chromium 内核浏览器 (如 Chrome, Edge)。
  • 您需要下载 BILIBILI-EX 的 .crx 格式安装文件。

下载 .crx 文件 ​

请从以下官方渠道下载最新或您需要的 .crx 文件：

点击下载 BILIBILI-EX 离线版 crx 文件

  • 官方发布页面: 下载页面
    • 通常文件名为 latest.crx.zip, 解压后为 bilibilia-x.x.x.crx。

将下载好的 .crx 文件保存到您容易找到的位置。

安装步骤 ​

  1. 打开浏览器扩展页面:

    • Chrome: 在地址栏输入 chrome://extensions/ 并按 Enter。
    • Edge: 在地址栏输入 edge://extensions/ 并按 Enter。

  2. 启用开发者模式: 在打开的扩展管理页面右上角，找到并 启用“开发者模式” (Developer mode) 的开关。

    • Chrome 示例:
    • Edge 示例:

  3. 拖放安装 .crx 文件:

    • 打开您保存 .crx 文件的文件夹。
    • 将 .crx 文件 直接拖拽 到刚才打开的浏览器扩展管理页面上。

  4. 确认安装: 浏览器会弹出一个确认窗口，提示您添加该扩展程序。点击 “添加扩展程序” (Add extension) 或类似按钮。

  5. 安装完成: 安装成功后，BILIBILI-EX 会出现在您的扩展列表中，并且通常会在浏览器工具栏显示图标。您可以根据需要禁用开发者模式（但插件仍会保留）。

  6. 开始使用: 现在您可以访问 Bilibili (bilibili.com) 网站，BILIBILI-EX 插件将自动生效！

故障排除:

  • 无法拖放: 如果拖放无效，请确认开发者模式已启用。部分浏览器版本可能已禁用此功能，请尝试搜索适用于您浏览器版本的特定离线安装方法。
  • “程序包无效”错误: 这可能表示 .crx 文件已损坏或与您的浏览器版本不兼容。请尝试重新下载或检查浏览器兼容性。

如果您在安装过程中遇到其他问题，请查阅 常见问题解答 或通过 联系我们 页面寻求帮助。
